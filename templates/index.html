About

Tamizh is aimed at Tamil speakers to capture their language speech via this web solution and translate to corresponding text within the Tamil syllabic orthography.

This is an essential to educate the speakers of the language, engage them on how to represent their language in textual format using this platform.

Future Work

Future work on this project would be the possible development of a mobile application for increased ease of use and expansion of the currently limited word bank/vocabulary of 28 words (see appendix) to be able to detect any spoken Tamil word. Thus, building a more robust Speech to Text translation software.

Furthermore, in the future, a subscription-based learning approach will be implemented for individuals and educational institutions. To increase the popularity of this application and encourage engagement, the Tamil app will be advertised via social channels, encouraging friends to interact with the software. And finally, to improve user experience and make the learning process exciting, the platform will be gamified, by creating a quiz-based section of the application, where users will be asked to guess which one of the spoken words is the actual text from the list of texts displayed.
